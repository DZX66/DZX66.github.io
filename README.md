# 柳下回声

基于 **EUW**（自定义标记语言）的个人博客网站，使用 Vite 构建，托管于 GitHub Pages。

> 99.9% 的代码使用 vibe coding ✨

访问：[https://dzx66.github.io/](https://dzx66.github.io/)

## 技术栈

- Vite
- 原生 JavaScript / HTML / CSS
- EUW (Markdown + HTML + 内联模板)

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 部署

推送到 `gh-pages` 分支或使用 GitHub Pages 从 `dist` 目录部署。
