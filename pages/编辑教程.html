编辑教程

官方文档

这个教程会指导如何编辑页面并提交。

准备工具

vscode

点击这个链接，点击“Download for Windows”，下载安装包，如果提示“VSCode...exe可能会损害你的设备。是否仍要保留？”，点击“保留”。然后点击“打开文件”。如果出现“安全警告”，点击“运行”。选择“我同意此协议”，点击“下一步”，点击“浏览”，选择一个安装路径（不要装在C盘），点击“下一步”，默认即可，点击“下一步”，继续默认（全部勾选），点击“下一步”，等待安装，然后点“完成”。打开vscode，右下角会有一个“安装语言包”的弹窗，点击“安装并重启”，（如果没有，请点击左侧四个方块，搜索“简体中文”，选择第一个下载并安装(install)，然后右下角选择应用并重启）自此vscode安装完成。视频教程

克隆存储库

点击左侧第三个“源代码管理”，选择“下载适用于Windows的git”，点击“打开”，根据你的操作系统位数选择“32-bit Git for Windows Setup.”或“64-bit Git for Windows Setup.”，打开安装包，点击“Next”，点击“Browse...”选择一个安装路径，点击“Next”，再点击“Next”，再点击“Next”，选择“Use Visual Studio Code as Git's default editor”，点击“Next”，一直点击“Next”，直到开始安装，安装完毕后，点击“Finish”，回到vscode，点击左下角齿轮，选择“设置”，搜索“git.path”，选择“在 setting.json中编辑”，输入git安装的路径（如D:/Git/cmd/git.exe），ctrl+s保存，点击左侧“重新加载”，点击“克隆仓库”，点击“从GitHub克隆”，在打开的页面中点击“Create an account”，创建一个账户（人机验证过难可以尝试换设备或浏览器尝试），验证邮箱后点击下方的“Skip personalization”，回到vscode登录界面（你可能需要重新在vscode中打开），登录，点击“Authorize Visual-Studio-Code”，登录（若失败，请在右下角选择用其他方式登录），在弹窗的框里输入“dzx66.github.io”，点击第一个搜索结果，选择一个文件夹作为工作目录，点击“选择为存储库目标”，克隆完后，点击“打开”，勾选“信任父文件夹...”，点击“是，我信任此作者”，自此，所有准备工作已完成。

打开编辑程序

找到system/main.exe，打开它，就可以对网站进行编辑。

上传更新

所有更改完成后，选择后台程序的“部署网站”，部署完成后，点击左侧“源代码管理”，在“消息”填写这次更改改了什么东西，请务必详细，然后点提交。
